# Care-Transition-Efficiency-Analytics

Overview
This project analyzes the efficiency of the Unaccompanied Children (UAC) care pipeline from CBP custody to HHS care and final sponsor placement.

## Objectives

* Measure CBP to HHS transfer efficiency
* Analyze discharge effectiveness
* Detect care pipeline bottlenecks
* Evaluate placement outcome stability

## Technologies

* Python
* Pandas
* Plotly
* Streamlit

## Key KPIs

* Transfer Efficiency Ratio
* Discharge Effectiveness Index
* Pipeline Throughput Rate
* Backlog Accumulation Rate
* Outcome Stability Score

## Dashboard Features

* Pipeline Flow Visualization
* Bottleneck Detection
* Trend Analysis
* KPI Monitoring
* Date Range Filtering
